Sobre mim

Projetos

Ao longo do meu periodo de estudos na área da tecnologia, principalmente no front-end, eu desenvolvi alguns projetos, são bem simples, uma vez que foram realizados apenas com o objetivo de ganhar experiência no assunto. A seguir estão os dois projetos que desenvolvi e algumas informações sobre cada um deles.

Projeto de notícias sobre minecraft.

Este projeto foi realizado como uma atividade avaliativa para um curso de HTML e CSS que realizei em 2021. O projeto é bastante simples, e sequer possui responsividade já que eu ainda era bastante inesperiente quando o desenvolvi. Se trata de um site de notícias sobre um jogo eletrônico o qual gosto bastante, Minecraft, atualmente ele contém apenas três notícias e uma página de feedback e foi desenvilvido inteiramente em HTML e CSS, sem uso de JavaScript.

Projeto Patinhas Douradas.

Este é o projeto mais recente que desenvolvi, feito em abril de 2024. Patinhas Douradas é o nome dado a um site ficticio de adoção de animais, o objetivo por trás deste projeto era desenvolver algo que pudesse ser utilizado de forma que ajudasse na causa do abandono de animais. O modelo ainda é bastante simples, porém, este já conta com adaptações de responsividade para diversos dispositivos.
